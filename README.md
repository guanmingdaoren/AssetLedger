# 设备资产台账

一个使用 PySide6 构建的本地桌面资产管理程序。所有设备当前信息、分类字典、系统设置和字段级变更历史均保存在同一个 Excel 工作簿中。

## 核心能力

- 每台设备拥有独立的 `AST-年份-序号` 资产UID，由系统自动生成且不可手动修改。
- `资产唯一标识符` 是可手动填写的业务字段，可为空；填写后必须保持唯一。
- 新增、编辑、搜索、排序以及按类别、使用状态、存放地点、品牌、取得方式筛选。
- 仅设备器材名称必填；bm 编码可为空，填写后必须保持唯一。
- 管理部门、管理人、使用部门、使用人支持从已有台账选择或直接输入。
- 一台设备支持记录任意数量的 SSD、机械硬盘或其他存储介质。
- 修改使用状态、存放地点、金额和存储介质等内容时自动记录修改前后值。
- 变化记录支持按单个变化字段查看；保存后立即刷新并展示最新历史。
- 启动时一次性读取 Excel 到内存，搜索、筛选、详情和历史查询不重复打开工作簿。
- 每次写入前自动备份，保存时使用临时文件原子替换。
- 检测程序运行期间的外部 Excel 修改，发现冲突时禁止覆盖并要求刷新。
- 设置页维护分类、存放地点、使用状态、取得方式和当前操作者。
- Excel 文件可直接打开查看，正式修改建议通过程序完成。

## 运行

需要 Python 3.11 或更高版本。

```powershell
python -m pip install -r requirements.txt
.\run_asset_ledger.bat
```

也可以直接运行：

```powershell
$env:PYTHONPATH="src"
python -m asset_ledger.app
```

首次启动会在项目目录下创建 `data/设备资产台账.xlsx`。之后可以在设置页切换到其他工作簿。

## 日常使用

1. 点击“新增设备”，填写设备器材名称和需要记录的其他信息。
2. 在主列表使用搜索框或左侧条件查找设备。
3. 选择设备后，右侧可查看完整信息和变化记录。
4. 双击设备或点击“编辑”修改；填写修改说明可帮助以后追溯。
5. 在“设置”中维护分类、存放地点、使用状态、取得方式和当前操作者。

状态为停用的分类或存放地点不会再出现在新建设备选项中，但已有设备记录仍保留并可查看。

程序运行期间的数据查询来自最近一次成功加载的内存缓存。若使用 Excel 或其他程序直接
修改了工作簿，请回到台账程序点击“刷新”后再继续保存；检测到外部修改时，程序会禁止
保存，避免覆盖外部内容。

## 数据安全与同步

- Excel 是唯一正式数据源，内存缓存只用于加速查询。
- 程序启动、点击“刷新”或成功保存后，会从正式 Excel 重建完整缓存。
- 每次实际写入前都会创建备份；设备编辑未发生内容变化时不会执行写入或创建备份。
- 保存期间使用临时文件并原子替换正式工作簿，降低中断导致文件损坏的风险。
- 如果程序检测到 Excel 被外部修改、删除、移动或占用，会停止保存并显示原因。
- 如果提示“Excel 已保存，但缓存刷新失败”，请勿重复新增或编辑；先点击“刷新”或重新启动程序。

## Excel 工作簿

工作簿包含以下工作表：

- `资产台账`：每台设备一行，保存当前信息。
- `存储介质`：每块存储介质一行，通过资产UID关联设备。
- `变更历史`：只追加字段级历史记录，并通过资产UID关联设备。
- `分类字典`：一级和二级设备分类。
- `位置字典`：位置层级。
- `枚举字典`：使用状态与取得方式选项。
- `系统配置`：工作簿版本、下一个资产序号和默认操作者。

每次写入前，程序会在工作簿旁的 `backups` 文件夹创建时间戳备份。

旧版工作簿第一次由新版程序打开时，会先自动备份，再升级字段结构并新增
`存储介质` 工作表。升级过程中不要同时使用 Excel 打开该文件。

## 打包 Windows 文件夹版

文件夹版会生成在 `dist/设备资产台账/`。其中：

- `设备资产台账.exe`：程序入口。
- `data/设备资产台账.xlsx`：首次启动后自动创建的业务工作簿。
- `_internal/`：程序运行依赖，不要删除或单独移动 EXE。
- `使用说明.md`：项目使用说明。

EXE、程序窗口和任务栏图标来自 `assets/app_icon.ico`。替换图标后需要重新运行打包脚本。

首次打包前安装构建依赖：

```powershell
python -m pip install -r requirements-build.txt
```

然后双击 `build_windows.bat`，或在 PowerShell 中运行：

```powershell
.\build_windows.bat
```

打包完成后运行：

```powershell
.\dist\设备资产台账\设备资产台账.exe
```

首次运行会在 EXE 同级的 `data` 文件夹中创建 `设备资产台账.xlsx`。发布程序时应复制整个
`dist/设备资产台账/` 文件夹；仅转发台账时，可以单独转发 Excel 文件。不要把真实业务
Excel 放入待发布文件夹后再发送给无关人员。
